A premium-looking GitHub profile viewer redesigned with glassmorphism (soft glass) UI using TailwindCSS, Framer Motion, and Recharts.

## What changed
- Project renamed to **GitHub Profile Viewer**
- Soft glass UI with frosted cards and subtle blur
- Improved typography, spacing and micro-interactions
- Minimal sidebar and topbar for a clean developer-focused layout

## Run locally
1. Install:
```bash
npm install
```
2. Run:
```bash
npm run dev
```

## Optional
For higher GitHub API limits and pinned repos, set `VITE_GITHUB_TOKEN` in `.env` or use `localStorage.setItem('GITHUB_TOKEN', 'ghp_xxx')`.
